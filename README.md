### Study it's a website planned to answer questions and made points, where i builted with Python and Django for Back-end and HTLM 5, CSS 3, JavaScript for Front-End.

## Technologies used:

* Python 3.9.11
* Django 4.2.3
* HTML 5
* CSS 3
* JavaScript ECMAScript 2020
* SQLite

## HOW TO EXECUTE THE PROJECT:

1️⃣ You need have Python and Django versions used in the project.<br>
2️⃣ Open the shell of your OS.<br>
3️⃣ Run in shell at the same folder in the project the code:<br>

* **Linux:** `python3 manage.py runserver`
* **Windows:** `py manage.py runserver`

4️⃣ Open in your browser end enjoy :)
